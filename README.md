# 🎨 Funky Clothing Hub

## Your Interactive Clothing Preference Website

Welcome to the Funky Clothing Hub! A beautiful, interactive, and animated website that helps you discover clothing based on your personal preferences.

### ✨ Features

- **Interactive Quiz**: Choose your style, color preference, and vibe
- **Smart Recommendations**: Get 15+ clothing items tailored to your preferences
- **Stunning Animations**: Smooth transitions and delightful micro-interactions
- **Pastel Aesthetic**: Soft, calming colors with a modern twist
- **Responsive Design**: Works perfectly on desktop, tablet, and mobile
- **Product Modal**: Click any product to see detailed information
- **Smooth Navigation**: Easy flow from landing → preferences → products

### 🎯 How It Works

1. **Landing Page** - Start with a beautiful, animated introduction
2. **Preference Quiz** - Select your:
   - Style (Casual, Formal, Sporty, Vintage)
   - Color (Pastel, Vibrant, Neutral, Dark)
   - Vibe (Minimalist, Bold, Artistic, Cozy)
3. **Product Display** - See 15 clothing items matched to your preferences
4. **Product Details** - Click any item to see full details in a modal

### 🛠️ Technology Stack

- **HTML5** - Semantic structure
- **CSS3** - Advanced animations and gradients
- **Vanilla JavaScript** - Pure JS, no frameworks
- **Responsive Design** - Mobile-first approach

- **Vibes**: Minimalist, Bold, Artistic, Cozy

### 🔄 Preference Matching

The matching algorithm works as follows:
1. Find exact matches (all 3 preferences match)
2. Find partial matches (at least 2 preferences match)
3. Show all products if no matches found
